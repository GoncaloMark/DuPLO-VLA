# DuPLO-VLA

This is the code for the thesis on VLMs and their application in robotics.

## Layout

Inside `/src` there are the notebooks used to produce datasets and run experiments on Google Colab. There is also the code for the components, LatentEncoder, Diffusion Policy, and Planner integrated with the VLM.

Under `/experiments` you'll find  2 dirs, one for libero and one for push_t, with the results and 1 video.

## Datasets

Available at: [HF](https://huggingface.co/lnxdre4d/datasets)
